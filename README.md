## ✨ 功能特性
- 网页 URL 安全扫描模拟
- 实时扫描进度 + 动态日志
- 恶意脚本、钓鱼链接、病毒特征检测
- 网络活动可视化图表
- 系统感染全屏警告特效
- 3 分钟倒计时整蛊
- 鼠标轨迹、病毒粒子、屏幕抖动、故障风动画
- 纯前端实现，无需后端

---

## 🚀 一键部署（GitHub Pages）
1. Fork 本仓库
2. 进入 Settings → Pages
3. Source 选择 main 分支
4. 等待 1 分钟即可生成在线链接

---

## 🎮 使用方法
1. 打开页面
2. 输入任意网址开始扫描
3. 等待扫描进度完成
4. 自动触发系统感染整蛊效果
5. 体验弹窗、倒计时、屏幕抖动等特效
6. 去整蛊同学
